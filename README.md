# 스터디 규칙

* 모이는 날 참여 불가능하면 미리 말하기(단, 경조사 등이 아니라 단순히 준비하기 싫어서는 안됨)
  * 인원이 적으면 날짜를 아예 바꾸기
* 무단으로 불참하면 스터디 제외
* 자료(발표자료, 정리자료, 코드 등)는 발표전에 깃에 올리기
  * 주제
  * 딥러닝>210616자연어처리.md
* 발표자료는 markdown
* 질의응답 때 시간이 너무 오래걸리거나 중요도가 낮은 질문에 대해선 발표자 재량으로 넘어갈 수 있다.
* 발표주제는 단순히 수업내용을 복습하는 것은 안된다(새로운 것이어야 함)
